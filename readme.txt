Short Description: A plugin to track user clicks on specific HTML elements using unique IDs or classes.

== Description ==
The **IDClass Click Counter** plugin allows you to generate unique IDs or classes for HTML elements and track user clicks on these elements from the frontend. You can view the click statistics in the WordPress dashboard.

== Features ==
- Generate class or ID selectors.
- Track clicks on frontend elements.
- View click statistics on the admin settings page.
- Add or delete elements dynamically.

= PRO Features =
- Track numbers of impressions on frontend elements.
- Track duration of impressions on frontend elements.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/idclass-click-counter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to "Click Counter" in your dashboard to start adding elements.

== Frequently Asked Questions ==

= What is the purpose of this plugin? =
This plugin allows you to track the number of clicks on specific frontend elements by adding unique class or ID selectors.


= How can I reset the click counts? =
To reset the click count for any element, delete the element from the admin page and re-add it.
